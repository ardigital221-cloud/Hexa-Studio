# ARDigital — AI-Агентство

Лендинг для AI-агентства с **Zero-Gravity** физикой на Matter.js.

## 🚀 Возможности

- **Zero-Gravity Physics** — плавающие элементы с отталкиванием от курсора мыши (Matter.js)
- **Dark Theme** — кибер-минимализм с неоновыми акцентами
- **GSAP Анимации** — плавное появление блоков при скролле (ScrollTrigger)
- **Адаптивность** — полная поддержка мобильных устройств
- **Интеграции** — заглушки для WhatsApp и Telegram

## 🛠 Стек

- HTML5 + CSS3 + Vanilla JavaScript
- [Matter.js](https://brm.io/matter-js/) — физика
- [GSAP + ScrollTrigger](https://gsap.com/) — анимации

## 📁 Структура

```
ardigital/
├── index.html              — Главный HTML
├── css/
│   └── styles.css          — Стили
├── js/
│   ├── matter-physics.js    — Matter.js Zero-Gravity
│   ├── gsap-animations.js  — GSAP ScrollTrigger
│   └── main.js             — Инициализация
└── README.md
```

## 🚀 Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
# или
python -m http.server 8000
```
